# ESG Performance Analytics Tool
A Streamlit-based interactive application for analyzing company ESG (Environmental, Social, Governance) scores using stock ticker input.

## Features
- Input stock ticker (e.g., AAPL, TSLA, XOM, BTI)
- Display E / S / G / Total ESG scores
- Show ESG trend chart (2020–2024)
- Risk level evaluation: Low / Medium / High Risk
- Data table for historical ESG performance
- Clean and professional UI

## Data Source
Simulated ESG data for educational and academic use (equivalent to Wind / Bloomberg for assignment purpose).

## How to Run Locally
### 1. Install required packages
```bash
pip install streamlit pandas numpy matplotlib
### 2. Run the application
```bash
streamlit run esg_app.py
